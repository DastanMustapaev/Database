# Database Course - Spring 2026

This repository contains my work for the Database course (Spring 2026).  
The labs cover PostgreSQL basics, querying, transactions, joins, aggregate functions, and backup/restore techniques.

## Labs

### Lab 3  
Basic psql commands and simple SQL operations.

### Lab 4  
SELECT queries and filtering data.

### Lab 5  
Working with CREATE DATABASE and DROP DATABASE.

### Lab 6  
Tables, data types, and constraints.

### Lab 7  
Primary keys and unique identification.

### Lab 8  
Foreign keys and relationships.

### Lab 9  
Database design basics and normalization.

### Lab 10  
Viewing database and table structure.

### Lab 11  
Basic data operations (INSERT, UPDATE, DELETE).

### Lab 12  
Querying data with SELECT, WHERE, CASE, and CTE.

### Lab 13  
Aggregate functions and GROUP BY.

### Lab 14  
Joining tables with INNER, LEFT, RIGHT, and FULL OUTER JOIN.

### Lab 15  
Advanced querying with subqueries, CTEs, window functions, and UNION.

### Lab 16  
Transactions and ACID properties.

### Lab 17  
Data import/export using COPY and backup/restore with pg_dump and pg_restore.

## Tools

- PostgreSQL
- pgAdmin 4
- SQL
